#Urban Traffic Flow Pattern Analysis

#Overview

This project analyzes urban traffic patterns using Exploratory Data Analysis (EDA).
Synthetic data is generated to simulate real-world traffic scenarios.

#Features
* Synthetic traffic data generation
* Data preprocessing (duplicates + missing values handling)
* Statistical analysis (mean, median, skewness, etc.)
* Hour-wise, day-wise, and monthly analysis
* Heatmaps and location-based insights

#Key Insights
* Peak traffic occurs during **7–9 AM and 5–7 PM**
* Weekday traffic is significantly higher than weekends
* Traffic distribution is **right-skewed**
* Night traffic is minimal
* Highway B shows highest traffic volume

#Tech Stack
* Python

#Conclusion
EDA reveals strong temporal patterns in traffic, which can be useful for urban planning and congestion control.
